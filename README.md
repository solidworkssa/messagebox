# MessageBox

Encrypted on-chain messaging service.

## Features
- Multi-chain support (Base & Stacks)
- Secure Contract Logic
- React/Next.js Frontend

## Contracts
- Base: `contracts/base/src/MessageBox.sol`
- Stacks: `contracts/stacks/contracts/messagebox.clar`

## Setup
```bash
pnpm install
pnpm dev
```
